# Data Visualization Projects
## Projects

### Dice Rolling Visualization

This project simulates rolling one or more dice thousands of times and visualizes the frequency of each possible outcome using interactive bar charts.

Files:
- `die.py`
- `die_visual.py`
- `dice_visual.py`
- `dice_visual_d6d10.py`

### Random Walk Visualization

This project generates random walks and visualizes the paths using scatter plots.

Files:
- `random_walk.py`
- `rw_visual.py`

## Technologies

- Python 3
- Plotly
- Matplotlib

## How to Run

Dice visualization:

```bash
python dice_visual.py
```

or

```bash
python dice_visual_d6d10.py
```

or

```bash
python die_visual.py
```

Random walk visualization:

```bash
python rw_visual.py
```
